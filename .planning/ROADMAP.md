# Roadmap — AI-Powered Enterprise PPM

## Current Milestone: M1 — Foundation & Assessment Engine

### Phase 1: Authentication Module ⬜
- Register API, Login API, JWT generation/verification
- Current user dependency, RBAC middleware
- Auth router, schemas, service layer

### Phase 2: Assessment Module Completion ⬜
- Submit Response API, Calculate Scores, Generate Maturity Levels
- Complete assessment service layer
- Assessment detail/list endpoints

### Phase 3: AI Insights Engine ⬜
- AI Assessment Analysis, Recommendations
- Executive Summary generation, Maturity Report
- OpenAI/Gemini integration layer

### Phase 4: Portfolio Management ⬜
- Portfolio, Program, Project models
- CRUD endpoints, KPI tracking
- Portfolio dashboard data

### Phase 5: Risk Management ⬜
- Risk, Issue, Mitigation Plan models
- Risk matrix scoring, escalation workflows
- Risk dashboard endpoints

### Phase 6: Resource Management ⬜
- Resource, Skill, Allocation models
- Capacity planning algorithms
- Resource utilization endpoints

### Phase 7: Frontend Application ⬜
- Auth pages (login, register)
- Assessment wizard UI
- Dashboard with charts
- Portfolio/Risk/Resource management pages
